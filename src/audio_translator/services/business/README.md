# 业务服务 (Business Services)

此目录包含应用程序的业务逻辑服务组件，负责实现核心业务功能。

## 包含的服务

- **AudioService**: 音频服务，负责音频文件处理和分析
- **TranslatorService**: 翻译服务，负责音频内容的翻译
- **CategoryService**: 分类服务，负责文件分类管理
- **ThemeService**: 主题服务，负责UI主题管理
- **UCSService**: 统一通信服务，提供UCS标准解析功能

## 职责

- 实现音频处理和分析功能
- 提供音频内容翻译服务
- 管理文件分类和标签
- 处理UI主题和样式配置
- 提供标准化通信协议支持
